## Objective

Implement Eigenvector Centrality - measures node importance based on connections to other important nodes. Similar to PageRank but without damping factor. Google's original algorithm precursor.

## Details

### Type
- [x] Feature - New functionality or enhancement  

### Priority
- [x] P3 - Low (when time permits)

### Cypher Syntax
```cypher
RETURN eigenvectorCentrality()
RETURN eigenvectorCentrality(iterations)
```

### Return Format
```json
[
  {"node_id": 1, "user_id": "alice", "score": 0.42}
]
```

### Complexity
- O(V + E) per iteration
- Typically converges in 20-100 iterations

### Notes
- Power iteration method
- Similar to PageRank without damping
- May not converge for disconnected graphs
