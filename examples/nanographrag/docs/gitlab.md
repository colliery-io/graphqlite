# GitLab Inc.

GitLab is a web-based DevOps platform that provides a complete CI/CD toolchain in a single application. The company is known for its all-remote culture.

## History

GitLab began in 2011 when Ukrainian developer Dmitriy Zaporozhets created it. Sid Sijbrandij discovered the project in 2012 and co-founded GitLab Inc.

## Leadership

- Sid Sijbrandij, Co-founder and CEO
- Dmitriy Zaporozhets, Co-founder and Engineering Fellow

## Platform Features

### Source Code Management
- Git repository hosting
- Code review and merge requests
- Branch protection

### CI/CD
- Built-in continuous integration
- Continuous delivery pipelines
- Auto DevOps

### Security
- Static Application Security Testing (SAST)
- Dynamic Application Security Testing (DAST)
- Container scanning
- Dependency scanning

### DevOps Platform
- Issue tracking
- Wiki documentation
- Package registry
- Infrastructure as Code

## Business Model

### Tiers
- Free tier (open core model)
- Premium (team features)
- Ultimate (enterprise security)

### Self-Managed and SaaS
Available as self-hosted or GitLab.com cloud service.

## All-Remote Culture

GitLab pioneered all-remote work with no offices, documenting extensive remote work practices publicly.

## Competition

GitLab positions itself as a complete DevOps platform competing with GitHub, Bitbucket, and various point solutions.
