# Datadog, Inc.

Datadog is an American cloud monitoring and security platform company headquartered in New York City. The company provides observability for cloud-scale applications.

## History

Founded in 2010 by Olivier Pomel and Alexis Lê-Quôc. Both founders previously worked at Wireless Generation, where they experienced the challenges of monitoring distributed systems.

## Leadership

- Olivier Pomel, Co-founder and CEO
- Alexis Lê-Quôc, Co-founder and CTO

## Products

### Infrastructure Monitoring
- Cloud infrastructure monitoring
- Container monitoring
- Serverless monitoring
- Network monitoring

### Application Performance Monitoring (APM)
- Distributed tracing
- Continuous profiling
- Error tracking

### Log Management
- Log collection and analysis
- Log-based metrics

### Security
- Cloud Security Posture Management (CSPM)
- Application Security
- Cloud Workload Security

### Real User Monitoring
- Browser and mobile app monitoring
- Session replay

## Technology

### Agent-Based Architecture
Datadog uses lightweight agents for data collection.

### Unified Platform
All monitoring data in a single platform with correlated views.

## Competition

Datadog competes with Splunk, New Relic, Dynatrace, and cloud provider monitoring tools.

## Market Position

Datadog has become the leading independent cloud monitoring vendor, with strong growth in enterprise adoption.
