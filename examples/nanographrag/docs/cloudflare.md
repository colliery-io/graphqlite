# Cloudflare, Inc.

Cloudflare is an American web infrastructure and security company headquartered in San Francisco. The company provides content delivery network, DDoS mitigation, and distributed DNS services.

## History

Founded in 2009 by Matthew Prince, Lee Holloway, and Michelle Zatlyn. The company launched publicly at TechCrunch Disrupt in 2010.

## Leadership

- Matthew Prince, Co-founder and CEO
- Michelle Zatlyn, Co-founder and President

## Products

### Performance
- Content Delivery Network (CDN)
- Argo Smart Routing
- Image optimization
- Caching

### Security
- DDoS protection
- Web Application Firewall (WAF)
- Bot management
- Zero Trust access (Cloudflare Access)
- Email security

### Developer Platform
- Cloudflare Workers (serverless compute)
- Pages (static site hosting)
- R2 (object storage)
- D1 (serverless SQL)
- Workers KV (key-value storage)

### Network Services
- DNS (1.1.1.1 resolver)
- WARP VPN
- Magic Transit

## Scale

Cloudflare's network spans over 300 cities and handles a significant percentage of global internet traffic.

## Privacy Focus

Cloudflare's 1.1.1.1 DNS resolver emphasizes privacy, not logging identifying information.

## Competition

Cloudflare competes with Akamai, Fastly, and AWS CloudFront in the CDN market, and increasingly with serverless platforms.
