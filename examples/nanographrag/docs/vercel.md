# Vercel Inc.

Vercel is a cloud platform for frontend developers that enables instant deployments and optimal performance. The company created Next.js, a popular React framework.

## History

Founded in 2015 by Guillermo Rauch as ZEIT. The company was renamed Vercel in 2020. Rauch previously created Socket.io and was CTO at LearnBoost.

## Leadership

- Guillermo Rauch, Founder and CEO

## Products

### Vercel Platform
- Instant deployments
- Preview deployments for branches
- Serverless functions
- Edge functions
- Analytics
- Image optimization

### Next.js
Open-source React framework developed by Vercel:
- Server-side rendering
- Static site generation
- API routes
- File-based routing
- Image optimization

### v0
AI-powered UI generation tool.

### Turbo
Build system for JavaScript/TypeScript monorepos (Turborepo).

## Technology

### Edge Network
Global edge network for fast content delivery.

### Serverless
Functions that deploy to edge locations automatically.

## Business Model

Free tier for hobby projects with paid tiers for professional use and enterprises.

## Competition

Vercel competes with Netlify, AWS Amplify, and cloud platforms for frontend deployment.

## Open Source

Next.js is one of the most popular React frameworks, with strong adoption in the React ecosystem.
