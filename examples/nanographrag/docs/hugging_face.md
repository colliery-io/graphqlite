# Hugging Face

Hugging Face is an AI company that develops tools for building applications using machine learning. The company hosts the largest open-source ML model repository.

## History

Founded in 2016 by Clément Delangue, Julien Chaumond, and Thomas Wolf. Originally a chatbot company, Hugging Face pivoted to become an ML platform.

## Leadership

- Clément Delangue, Co-founder and CEO
- Julien Chaumond, Co-founder and CTO
- Thomas Wolf, Co-founder and Chief Science Officer

## Products

### Hub
- Model repository (200,000+ models)
- Dataset repository
- Spaces (ML app hosting)
- Model cards and documentation

### Transformers Library
Open-source library for state-of-the-art NLP:
- Pre-trained models
- Fine-tuning pipelines
- Multi-framework support (PyTorch, TensorFlow, JAX)

### Other Libraries
- Datasets - data loading library
- Tokenizers - fast tokenization
- Diffusers - diffusion models
- Accelerate - distributed training

### Inference API
Hosted inference for models.

### AutoTrain
Automated model training.

## Community

Hugging Face has built a massive open-source community:
- Model sharing
- Paper implementations
- Educational content

## Business Model

Free platform with paid tiers for compute, enterprise features, and private hosting.

## Impact

Hugging Face has become essential infrastructure for AI/ML development, democratizing access to state-of-the-art models.
