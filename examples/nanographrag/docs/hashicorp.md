# HashiCorp, Inc.

HashiCorp is an American software company that provides infrastructure automation tools for multi-cloud environments. IBM acquired HashiCorp in 2024.

## History

Founded in 2012 by Mitchell Hashimoto and Armon Dadgar. The founders met at the University of Washington and began creating open-source tools.

## Leadership

- Armon Dadgar, Co-founder and CTO
- Mitchell Hashimoto, Co-founder (stepped back from day-to-day 2021)
- David McJannet, CEO

## Products

### Terraform
Infrastructure as Code (IaC) tool for provisioning cloud resources. The most widely used IaC tool.

### Vault
Secrets management and encryption platform.

### Consul
Service networking and service mesh solution.

### Nomad
Workload orchestrator, alternative to Kubernetes.

### Boundary
Identity-based access management.

### Waypoint
Application deployment platform.

### Packer
Machine image building tool.

### Vagrant
Development environment management.

## Business Model

HashiCorp operates open-source projects with enterprise versions offering additional features, support, and cloud hosting.

## Licensing Change

In 2023, HashiCorp changed Terraform and other products from MPL to BSL (Business Source License), sparking community debate and the OpenTofu fork.

## IBM Acquisition

IBM announced the acquisition of HashiCorp in 2024 for $6.4 billion, integrating it into IBM's hybrid cloud strategy.
