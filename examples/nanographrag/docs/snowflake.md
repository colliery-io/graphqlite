# Snowflake Inc.

Snowflake Inc. is an American cloud computing company that provides a cloud-based data warehousing platform. The company is headquartered in Bozeman, Montana, with additional offices in San Mateo, California.

## History

Snowflake was founded in 2012 by Benoit Dageville, Thierry Cruanes, and Marcin Zukowski, all former Oracle engineers. The company's IPO in September 2020 was the largest software IPO in history at the time.

## Leadership

- Sridhar Ramaswamy, Chief Executive Officer (since 2024)
- Frank Slootman, former CEO (2019-2024) and current Chairman
- Benoit Dageville, co-founder and President of Products
- Grzegorz Czajkowski, Executive Vice President of Engineering

## Platform

Snowflake's Data Cloud platform provides:

- Data warehousing
- Data lake capabilities
- Data engineering
- Data science and machine learning
- Data sharing and collaboration
- Application development (Snowpark)

The platform runs on Amazon Web Services, Microsoft Azure, and Google Cloud Platform.

## Business Model

Snowflake uses a consumption-based pricing model where customers pay for the compute and storage resources they use, rather than fixed licensing fees.

## Competition

Snowflake competes with:

- Amazon Redshift
- Google BigQuery
- Microsoft Azure Synapse
- Databricks
- Oracle Autonomous Data Warehouse

## Notable Investors

Warren Buffett's Berkshire Hathaway and Salesforce Ventures were notable investors in Snowflake's IPO.
