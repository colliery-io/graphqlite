# Perplexity AI

Perplexity AI is an AI-powered search engine that provides direct answers with cited sources. The company aims to revolutionize how people search for information.

## History

Founded in 2022 by Aravind Srinivas, Denis Yarats, Johnny Ho, and Andy Konwinski. Srinivas previously worked at Google AI and OpenAI.

## Leadership

- Aravind Srinivas, Co-founder and CEO
- Denis Yarats, Co-founder and CTO
- Johnny Ho, Co-founder and CSO
- Andy Konwinski, Co-founder

## Products

### Perplexity Search
AI-powered search that:
- Provides direct answers to questions
- Cites sources for verification
- Follows up with related questions
- Supports conversational queries

### Perplexity Pro
Premium tier with:
- More powerful models
- Unlimited Pro searches
- File upload and analysis
- API access

### API
Developer access to Perplexity's search capabilities.

## Technology

Perplexity combines:
- Large language models
- Real-time web search
- Source citation and verification
- Conversational interface

## Differentiation

Unlike ChatGPT, Perplexity:
- Always cites sources
- Focuses on search/research use cases
- Has real-time web access
- Emphasizes accuracy over creativity

## Competition

Competes with Google, Bing, and AI assistants like ChatGPT and Claude in the search and research space.

## Funding

Raised significant funding including from NVIDIA and Amazon's Jeff Bezos.
