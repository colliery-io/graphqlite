# Elastic N.V.

Elastic is a Dutch-American company that develops the Elastic Stack, including Elasticsearch, a distributed search and analytics engine.

## History

Elasticsearch was created by Shay Banon in 2010. Elastic was formally incorporated in 2012. The company went public in 2018.

## Leadership

- Ash Kulkarni, CEO (since 2022)
- Shay Banon, Founder and CTO

## Products

### Elastic Stack (ELK Stack)

#### Elasticsearch
Distributed search and analytics engine based on Apache Lucene.

#### Kibana
Visualization and management interface.

#### Logstash
Data processing pipeline.

#### Beats
Lightweight data shippers.

### Solutions

- Observability - logs, metrics, APM
- Security (SIEM)
- Enterprise Search

### Elastic Cloud
Managed Elasticsearch service on major cloud platforms.

## Use Cases

- Log analytics and monitoring
- Application performance monitoring (APM)
- Security analytics (SIEM)
- Site search
- Business analytics

## Open Source

Elasticsearch was originally Apache licensed. In 2021, Elastic changed to SSPL/Elastic License, leading Amazon to fork the project as OpenSearch.

## Market Position

Elastic competes with Splunk, Datadog, and cloud providers' native search and observability services.

## Developer Community

The Elastic community is large, with extensive documentation, training programs, and active forums.
