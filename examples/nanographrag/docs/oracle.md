# Oracle Corporation

Oracle Corporation is an American multinational computer technology company headquartered in Austin, Texas. The company specializes in database software, cloud systems, and enterprise software products.

## History

Oracle was founded on June 16, 1977, by Larry Ellison, Bob Miner, and Ed Oates as Software Development Laboratories. The company was renamed to Oracle in 1983 after its flagship product, the Oracle Database.

## Leadership

- Safra Catz, Chief Executive Officer
- Larry Ellison, Chairman and Chief Technology Officer (founder)

Larry Ellison is one of the wealthiest people in the world and remains deeply involved in Oracle's strategy and product development.

## Products

### Database
- Oracle Database (industry-leading relational database)
- MySQL (acquired via Sun Microsystems)
- Oracle Autonomous Database

### Cloud Infrastructure
- Oracle Cloud Infrastructure (OCI)
- Oracle Cloud Applications

### Enterprise Applications
- Oracle E-Business Suite
- Oracle Fusion Applications
- NetSuite (cloud ERP, acquired 2016)
- Cerner (healthcare IT, acquired 2022)

### Other
- Java (programming language, via Sun acquisition)
- VirtualBox (virtualization)

## Acquisitions

Major acquisitions include:

- Sun Microsystems (2010) for $7.4 billion
- NetSuite (2016) for $9.3 billion
- Cerner (2022) for $28.3 billion

## Competition

Oracle competes with:

- Microsoft in databases and cloud
- SAP in enterprise applications
- Amazon Web Services in cloud infrastructure
- Salesforce in cloud applications
- IBM in enterprise software

## Headquarters

Oracle relocated its headquarters from Redwood City, California to Austin, Texas in 2020.

## AI Partnerships

Oracle has partnered with NVIDIA for AI infrastructure and signed deals with OpenAI and Cohere to provide AI services on Oracle Cloud.
