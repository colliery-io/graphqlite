# Retool

Retool is a low-code platform for building internal tools. The company enables developers to quickly create admin panels, dashboards, and internal applications.

## History

Founded in 2017 by David Hsu while he was a student at Columbia University. Hsu previously worked at Palantir.

## Leadership

- David Hsu, Founder and CEO

## Product Features

### Builder
- Drag-and-drop interface builder
- Pre-built components (tables, charts, forms)
- Custom JavaScript and CSS
- Responsive layouts

### Integrations
Connects to various data sources:
- Databases (PostgreSQL, MySQL, MongoDB)
- APIs (REST, GraphQL)
- SaaS tools (Stripe, Salesforce, etc.)
- Cloud services (AWS, GCP)

### Workflows
Automated backend processes and scheduled jobs.

### Mobile
Build internal mobile apps with the same platform.

### Self-Hosted
Option to deploy on own infrastructure.

## Use Cases

- Admin dashboards
- Customer support tools
- Operations panels
- Approval workflows
- Data entry applications

## Business Model

Usage-based pricing with free tier for small teams.

## Target Market

Engineering teams building internal tools at startups and enterprises.

## Competition

Retool competes with Appsmith, Budibase, and traditional internal tool development approaches.

## Funding

Raised significant funding including a $45 million Series B and later rounds, reaching over $3 billion valuation.
