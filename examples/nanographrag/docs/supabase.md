# Supabase Inc.

Supabase is an open-source Firebase alternative providing a backend-as-a-service platform. The company is based in Singapore and San Francisco.

## History

Founded in 2020 by Paul Copplestone and Ant Wilson. The founders aimed to create an open-source alternative to Firebase using PostgreSQL.

## Leadership

- Paul Copplestone, Co-founder and CEO
- Ant Wilson, Co-founder and CTO

## Products

### Database
- Managed PostgreSQL databases
- Real-time subscriptions
- Database backups

### Authentication
- User management
- Social login providers
- Row-level security

### Storage
- File storage with CDN
- Image transformations

### Edge Functions
- Serverless functions using Deno

### Realtime
- Broadcast messages
- Presence tracking
- Database changes streaming

### Vector (pgvector)
- AI/ML embeddings storage
- Similarity search

## Technology

### PostgreSQL Foundation
Built on PostgreSQL with extensions:
- PostgREST for APIs
- GoTrue for auth
- Realtime for subscriptions

### Open Source
Core components are open source, allowing self-hosting.

## Business Model

Managed service with free tier and paid plans. Self-hosting is free.

## Competition

Supabase competes with Firebase, AWS Amplify, and other backend-as-a-service platforms.

## Developer Focus

Strong developer experience with client libraries for multiple languages and frameworks.
