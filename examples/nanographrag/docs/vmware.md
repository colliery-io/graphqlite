# VMware, Inc.

VMware is an American cloud computing and virtualization technology company. The company pioneered x86 virtualization and remains a leader in enterprise software.

## History

Founded in 1998 by Diane Greene, Mendel Rosenblum, Scott Devine, Edward Wang, and Edouard Bugnion. EMC acquired VMware in 2004, and it was later spun off.

## Ownership History

- EMC Corporation (2004-2016)
- Dell Technologies (2016-2021)
- Publicly traded (2007-2022)
- Broadcom (acquired 2023 for $61 billion)

## Leadership

- Hock Tan, CEO (via Broadcom acquisition)
- Previous CEOs: Raghu Raghuram, Pat Gelsinger, Paul Maritz, Diane Greene

## Products

### Virtualization
- vSphere (ESXi hypervisor)
- vCenter Server
- vSAN storage virtualization

### Cloud
- VMware Cloud on AWS
- VMware Cloud Foundation
- Tanzu (Kubernetes)

### End User Computing
- Workspace ONE
- Horizon VDI

### Networking
- NSX network virtualization

## Industry Impact

VMware revolutionized data centers by enabling multiple virtual machines on single physical servers, dramatically improving hardware utilization.

## Broadcom Acquisition

Broadcom acquired VMware in 2023 for $61 billion, transitioning it to a subscription-based model and integrating it into Broadcom's enterprise software portfolio.
