# Stability AI

Stability AI is a company focused on open-source generative AI. The company is best known for Stable Diffusion, an open-source text-to-image model.

## History

Founded in 2020 by Emad Mostaque. Stability AI gained prominence in 2022 with the release of Stable Diffusion.

## Leadership

- Emad Mostaque, Founder (stepped down as CEO in 2024)
- Leadership transitions ongoing

## Products

### Stable Diffusion
Open-source text-to-image model:
- Image generation from text prompts
- Image-to-image transformation
- Inpainting and outpainting
- Multiple versions (SD 1.5, SDXL, SD3)

### Other Models
- Stable Audio - music and sound generation
- Stable Video - video generation
- StableLM - large language models
- Stable Code - code generation

### DreamStudio
Web interface for Stable Diffusion.

### Developer APIs
API access to Stability models.

## Open Source Approach

Unlike competitors, Stability AI releases model weights openly, enabling:
- Local deployment
- Fine-tuning and customization
- Community innovation
- Research access

## Impact

Stable Diffusion sparked widespread adoption of generative AI for images, inspiring countless applications and research.

## Challenges

The company has faced challenges including:
- Funding concerns
- Leadership changes
- Competition from well-funded competitors
- Copyright debates around training data
