# Unity Technologies

Unity Technologies is an American video game software development company headquartered in San Francisco, California. The company is best known for the Unity game engine, one of the most widely used game development platforms.

## History

Unity Technologies was founded in 2004 in Copenhagen, Denmark, by David Helgason, Joachim Ante, and Nicholas Francis. The company later relocated its headquarters to San Francisco.

## Leadership

- Matthew Bromberg, Chief Executive Officer (since 2024)
- John Riccitiello, former CEO (2014-2023)
- Luis Felipe Visoso, Chief Financial Officer

## Products

### Unity Engine
A cross-platform game engine used to develop games for mobile, desktop, consoles, and VR/AR platforms. Unity powers over 50% of mobile games.

### Features
- Real-time 3D development
- Visual scripting (Unity Visual Scripting)
- Multiplayer services (Netcode)
- Cloud services (Unity Gaming Services)
- Asset Store (marketplace for tools and assets)

### Unity Ads
An advertising platform for mobile game monetization.

### Unity AI
Machine learning tools for game development.

## Controversy

In 2023, Unity faced significant backlash after announcing a "Runtime Fee" that would charge developers based on game installs. The company later revised the policy following developer outcry.

## ironSource Merger

Unity merged with ironSource, a mobile monetization company, in 2022 in a deal valued at $4.4 billion.

## Competition

Unity competes with Epic Games (Unreal Engine), Godot (open source), and various specialized game development tools.

## Market Position

Unity is the most widely used game engine by number of developers, particularly for mobile and indie game development.
