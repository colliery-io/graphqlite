# Linear

Linear is a software project management tool built for modern software teams. The company is known for its fast, keyboard-driven interface.

## History

Founded in 2019 by Karri Saarinen and Tuomas Artman. Both founders previously worked at Airbnb, where Saarinen led design systems and Artman led mobile engineering.

## Leadership

- Karri Saarinen, Co-founder and CEO
- Tuomas Artman, Co-founder and CTO

## Product Features

### Issue Tracking
- Fast, keyboard-first interface
- Real-time sync
- Markdown support
- Issue priorities and estimates

### Projects and Cycles
- Project management
- Sprint planning with Cycles
- Roadmaps
- Milestones

### Integrations
- GitHub/GitLab integration
- Slack integration
- Figma integration
- API access

## Design Philosophy

### Speed
Built for speed with local-first architecture and optimistic updates.

### Opinionated
Pre-configured workflows rather than endless customization.

### Aesthetic
Clean, modern design inspired by professional creative tools.

## Technology

Linear pioneered synchronization engine techniques that influenced other products, leading to the "local-first" movement in app development.

## Target Market

Software development teams, particularly at startups and tech companies.

## Competition

Linear competes with Jira, Asana, and other project management tools, positioning itself as the modern alternative.

## Growth

Linear has grown rapidly among tech companies, becoming particularly popular with startups and engineering teams.
